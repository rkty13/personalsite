• matrox

    Python

    Matrox is a linear algebra library currently capable of basic matrix operations, factorizations (LU, LUP, LDU, etc.), and Gaussian elimination. To write this, I am following Gilbert Strang's textbook, "Linear Algebra and Its Applications, 4th Edition" as well as concepts/methods I find on the internet. This is currently a side project that I continue to work on during my free time.

  • hackBCA III Website

    Flask
    Sendgrid API
    Twilio API
    MongoDB
    Amazon EC2/S3

    The hackBCA III website called for a complete ground-up rebuilding from the hackBCA II website. This meant converting to an MVC framework, modularizing many of the features of the backend, and enforcing a schema on our MongoDB database. Staff tools were also delegated to its own web application in order to keep the two different codebases separate. Keeping our code open-source was also very important, so all of our code for hackBCA III is available on Github.

  • Pi Git

    Raspberry Pi
    Flask

    Pi Git was created in order to solve an issue that many hackers face at hackathons: bad wifi. Because many people rely on github for collaboration, bad wifi prevents hackers from contributing to a project. Pi Git uses a Raspberry Pi to create a Git server that users can connect to via its own wifi network and be able to collaborate locally without an internet connection. This project won first place at IronHacks.

  • The Clean Sleep

    Flask
    Google Places API
    MongoDB
    Heroku

    The Clean Sleep was made at HackRU Fall 2014. It displays reviews of hotels based on their cleanliness. Users can create reviews of a hotel they visit which are stored in a MongoDB database, as well as viewing reviews that other people have made.

  • FactsMeUp!

    Flask
    Twilio API
    MongoDB
    Heroku

    FactsMeUp! was created at the Breaking Barriers Hackathon in August 2014. It allows users to send a text message of a fact to a desired phone number via the Twilio API. Users can also create and update lists of facts that others have created.

  • BBChat

    Java Swing
    Java Networking

    BBChat was created as a final project in our Java Programming class. Clients can connect to the server and communicate with other users in a global chat room. It also features customizable fonts/font sizes, chat noise, as well as a command feature.
